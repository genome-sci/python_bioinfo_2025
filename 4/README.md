# 4. Pythonを用いた基礎的なシングルセルRNA-seq解析　遺伝研　望月孝子

## 概要
本講習では、シングルセルRNA-seq の基礎的な、
 - 細胞のフィルタリング
 - 発現変動遺伝子の抽出
 - 次元削減とクラスタリング
を行います。

本講習では、基本的に2004年度の国立遺伝学研究所 東先生のシングルセルRNAseq解析の講習資料を
使って講義を進めます。
後続の解析にご興味がある方は、[2004年度の東先生の講習資料](https://github.com/genome-sci/python_bioinfo_2024/tree/main/2-2)をご参考ください。
